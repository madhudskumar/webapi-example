Device Orientation Api

This works with most of the browsers or you need to polyfill. Gives the Orientation with alpha beta and gama being x y and z parameters

Note: gyronorm.js is a polyfill for normalizing the accelerometer and gyroscope data on mobile devices. This is useful for overcoming some of the differences in device support for device orientation. This library depends on software with Creative Commons Non-Commercial License 4.0.
